// how it works

How the score works.

Reputation, not referrals. Every score is built from on-chain calls — no screenshots, no self-reporting.

01
Every call is tracked
Each on-chain call is pulled and scored on how far it actually ran after the call.
02
Size and recency weigh in
A big win on a real-cap token counts more than a micro-cap. Recent calls count fully; old ones fade.
03
Consistency beats luck
Repeat winners are rewarded. One-hit wonders and repeat ruggers get discounted.

Scores shown are the v19.8 reputation snapshot. Letter grades (A+ → D) land once the team locks the bands.
